# Translations

To improve translations for MCC, please visit: Crowdin - Minecraft Console Clientopen in new window.

It is recommended to translate MCC in-app text first.

If you can't find the language you want to translate into, please contact us at Github or Discord to add it.

Github: https://github.com/MCCTeam/Minecraft-Console-Client

Discord: https://discord.gg/9HPr2EE4C4
