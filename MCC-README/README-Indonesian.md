## **Tentang ℹ️**

**Minecraft Console Client (MCC)** is a lightweight cross-platform open-source Minecraft TUI client for **Java** edition that allows you to connect to any Minecraft Java server, send commands and receive text messages in a fast and easy way without having to open the main Minecraft game.

## Unduh

Dapatkan rilis terbaru dari [bagian Rilis](https://github.com/MCCTeam/Minecraft-Console-Client/releases/latest).

## Instal Cepat ⚡

Buka terminal di folder tempat Anda ingin menempatkan MCC dan jalankan:

Linux / macOS:

```bash
curl -fsSL https://mccteam.github.io/install.sh | sh
```

Windows (PowerShell):

```powershell
iwr -useb https://mccteam.github.io/install.ps1 | iex
```

Skrip ini mendeteksi arsitektur perangkat Anda dan mengunduh binary yang sesuai. Untuk opsi lainnya (termasuk `wget` dan unduhan manual), lihat [panduan instalasi](https://mccteam.github.io/guide/installation.html).

## Lisensi ⚖️

Kecuali disebutkan secara khusus, kode tersebut berasal dari Tim MCC atau Kontributor, dan tersedia di bawah lisensi CDDL-1.0. Jika tidak, lisensi dan penulis asli disebutkan dalam header file sumber. Prinsip-prinsip utama dari lisensi CDDL-1.0 pada dasarnya adalah sebagai berikut:

-   Anda dapat menggunakan kode berlisensi tersebut secara keseluruhan atau sebagian dalam program apa pun yang Anda inginkan, terlepas dari lisensi program tersebut secara keseluruhan (atau lebih tepatnya, dengan mengesampingkan kode yang Anda gunakan). Program itu sendiri bisa berupa open source atau closed source, gratis atau berbayar.
-   Ketika melakukan modifikasi, penyempurnaan, atau penambahan pada kode CDDL (setiap kode yang dirujuk dalam modifikasi langsung terhadap kode CDDL dianggap sebagai bagian dari kode CDDL dan karenanya tunduk pada persyaratan ini; misalnya, modifikasi suatu fungsi matematika untuk menggunakan tabel pencarian cepat menjadikan tabel tersebut sendiri sebagai tambahan pada kode CDDL, terlepas dari apakah tabel tersebut berada dalam file sumber tersendiri), semua perubahan tersebut wajib disediakan secara publik dan bebas dalam bentuk kode sumber, di bawah lisensi CDDL itu sendiri.
-   Dalam setiap program (baik sumber maupun biner) yang menggunakan kode CDDL, wajib diberikan pengakuan kepada sumber (baik proyek maupun penulis) dari kode CDDL tersebut. Selain itu, modifikasi terhadap kode CDDL (yang wajib didistribusikan dalam bentuk sumber) tidak boleh menghapus pemberitahuan yang menunjukkan asal-usul kode tersebut.
