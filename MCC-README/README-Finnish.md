## **Tietoja ℹ️**

**Minecraft Console Client (MCC)** on kevyt, monialustainen avoimen lähdekoodin Minecraft TUI -client **Java**-päivitystä varten, joka mahdollistaa yhteyden muodostamisen mihin tahansa Minecraft Java -palvelimeen sekä komentojen lähetystä ja tekstiviestien vastaanottamista nopeasti ja helposti ilman, että varsinaista Minecraft-peliä tarvitsee avata.

## Lataa

Hanki uusin versiojulkaisutiedoston</a kautta.</p> 



## Pikainstalliointi ⚡

Avaa pääte kansioon, johon haluat asentaa MCC:n, ja suorita:

Linux / macOS:



```bash
curl -fsSL https://mccteam.github.io/install.sh | sh
```


Windows (PowerShell):



```powershell
iwr -useb https://mccteam.github.io/install.ps1 | iex
```


Skripti tunnistaa laitearkkitehturisi ja lataa oikean binääritiedoston. Lisävaihtoehtoja (mukaan lukien `wget` ja manuaaliset lataukset) löytyy [asennusoppaasta](https://mccteam.github.io/guide/installation.html).

Ellei erikseen mainita, koodi on peräisin MCC-tiimiltä tai sen avustajilta ja se on saatavilla CDDL-1.0-lisenssin alaisena. Muussa tapauksessa lisenssi ja alkuperäinen tekijä on mainittu lähdetiedostojen otsikoissa. CDDL-1.0-lisenssin päätiedot ovat periaatteessa seuraavat:

-   Voit käyttää lisenssoitua koodia kokonaisuudessaan tai osittain missä tahansa haluamassasi ohjelmassa, riippumatta koko ohjelman (tai pikemminkin sitä lukuun ottamatta) käyttämästä lisenssistä. Itse ohjelma voi olla avoimen tai suljetun lähdekoodin, ilmaisen tai kaupallisen.
-   Kaikki muutokset, parannukset tai lisäykset CDDL-koodiin (kaikki koodi, jota käytetään suoraan muokattaessa CDDL-koodia, katsotaan CDDL-koodin lisäyksenä ja siten tähän vaatimukseen kuuluvaksi; esimerkiksi matematiikkafunktion muokkaaminen nopeaa hakupöytää käyttäväksi tekee kyseisestä pöydästä itsessään CDDL-koodin lisäyksen, riippumatta siitä, onko se omassa lähdekooditiedostossaan) on tehtävä julkisesti ja vapaasti saatavilla lähdetiedostoina itse CDDL-lisenssin alaisuudessa.
-   Kaikissa ohjelmissa (lähdekoodi- tai binäärimuodossa), joissa käytetään CDDL-koodia, on tunnustettava CDDL-koodin lähde (joko projekti tai kirjoittaja). Lisäksi CDDL-lisenssin mukaiseen koodiin tehtyjä muutoksia, joiden jakaminen on sallittu vain lähdekoodina, ei saa muuttaa niin, että koodin alkuperää koskevat ilmoitukset poistetaan.
