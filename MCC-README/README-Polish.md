## **Informacje ℹ️**

**Minecraft Console Client (MCC)** is a lightweight cross-platform open-source Minecraft TUI client for **Java** edition that allows you to connect to any Minecraft Java server, send commands and receive text messages in a fast and easy way without having to open the main Minecraft game.

## Pobierz

Pobierz najnowszą wersję z sekcji [Releases](https://github.com/MCCTeam/Minecraft-Console-Client/releases/latest).

## Szybka instalacja ⚡

Otwórz terminal w katalogu, w którym chcesz zainstalować MCC, i uruchom:

Linux / macOS:

```bash
curl -fsSL https://mccteam.github.io/install.sh | sh
```

Windows (PowerShell):

```powershell
iwr -useb https://mccteam.github.io/install.ps1 | iex
```

Skrypt wykrywa twoją architekturę i pobiera odpowiednią plik binarny. Aby uzyskać więcej opcji (w tym `wget` i ręczne pobieranie), zapoznaj się z [instrukcją instalacji](https://mccteam.github.io/guide/installation.html).
