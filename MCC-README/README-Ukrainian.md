## **Про ℹ️**

**Minecraft Console Client (MCC)** — це легкий багатоплатформенний відкритий клієнт із текстовим інтерфейсом для Java-видання **Minecraft**, який дозволяє швидко та просто підключатися до будь-якого сервера Minecraft Java, надсилати команди й отримувати текстові повідомлення, не відкриваючи головну гру Minecraft.

## Завантажити

Отримайте останній реліз у розділі [Релізи](https://github.com/MCCTeam/Minecraft-Console-Client/releases/latest).

## Швидка установка ⚡

Відкрийте термінал у папці, де ви хочете розмістити MCC, і виконайте команду:

Linux / macOS:

```bash
curl -fsSL https://mccteam.github.io/install.sh | sh
```

Windows (PowerShell):

```powershell
iwr -useb https://mccteam.github.io/install.ps1 | iex
```

Скрипт визначає вашу архітектуру та завантажує відповідний бінарний файл. Для отримання додаткових варіантів (зокрема, за допомогою `wget` та ручного завантаження) перегляньте [руководство з установки](https://mccteam.github.io/guide/installation.html).

## Ліцензія ⚖️

Це речення походить з Minecraft Console Client (MCC), текстового клієнта для Minecraft Java Edition. Зміст включає стрінги інтерфейсу користувача додатку, конфігурацію ботів та автоматизації, внутрішні команди, повідомлення про стан, а також документацію для користувачів, що охоплює інвентар, рельєф місцевості, сутності, крафт, переміщення, підключення до сервера й теми CLI/конфігурації. При перекладі надавайте перевагу офіційній термінології гри Minecraft. У разі, коли гравецька спільнота широко прийняла альтернативні терміни, обирайте найбільш впізнавані. Переклад має відповідати стилю цього клієнтського інструменту для Minecraft. Якщо не зазначено окремо, код належить команді MCC або співробітникам, які надали його за ліцензією CDDL-1.0. В іншому випадку ліцензія та початковий автор зазначені в заголовках вихідних файлів. Основні положення ліцензії CDDL-1.0 полягають у наступному:

-   Ви можете використовувати ліцензійний код повністю або частково у будь-якій програмі на ваш розсуд, незалежно від ліцензії самої програми загалом (або, точніше, за винятком самого запозиченого коду). Сама програма може бути відкритою або закритою, безкоштовною або комерційною.
-   Однак у будь-якому випадку всі зміни, покращення або доповнення до коду CDDL (будь-який код, на який посилаються під час безпосередніх змін у коді CDDL, вважається доповненням до цього коду і тому підпадає під цю вимогу; наприклад, зміна математичної функції для використання швидкісної таблиці пошуку робить саму цю таблицю доповненням до коду CDDL, незалежно від того, чи знаходиться вона в окремому файлі зсировинним кодом) повинні бути оприлюднені та безкоштовно надані у вихідному коді за самою ліцензією CDDL.
-   У будь-якій програмі (вихідному коді чи бінарному файлі), що використовує код під ліцензією CDDL, необхідно зазначати походження цього коду (проект або автор). Крім того, зміни до коду під ліцензією CDDL (який має поширюватися у вигляді вихідного коду) не повинні видаляти повідомлення, що вказують на походження коду.

Докладніша інформація на сайті http://qstuff.blogspot.fr/2007/04/why-cddl.html Повна ліцензія доступна за посиланням http://opensource.org/licenses/CDDL-1.0
