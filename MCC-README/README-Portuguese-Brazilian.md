## **Sobre ℹ️**

**Minecraft Console Client (MCC)** is a lightweight cross-platform open-source Minecraft TUI client for **Java** edition that allows you to connect to any Minecraft Java server, send commands and receive text messages in a fast and easy way without having to open the main Minecraft game.

## Baixar

Obtenha a versão mais recente na [seção de lançamentos](https://github.com/MCCTeam/Minecraft-Console-Client/releases/latest).

## Instalação Rápida ⚡

Abra um terminal na pasta onde você deseja instalar o MCC e execute:

Linux / macOS:

```bash
curl -fsSL https://mccteam.github.io/install.sh | sh
```

Windows (PowerShell):

```powershell
iwr -useb https://mccteam.github.io/install.ps1 | iex
```

O script detecta a sua arquitetura e baixa o binário correto. Para mais opções (incluindo `wget` e downloads manuais), consulte o [guia de instalação](https://mccteam.github.io/guide/installation.html).
