## **Tungkol sa ℹ️**

**Minecraft Console Client (MCC)** is a lightweight cross-platform open-source Minecraft TUI client for **Java** edition that allows you to connect to any Minecraft Java server, send commands and receive text messages in a fast and easy way without having to open the main Minecraft game.

## I-download

Kunin ang pinakabagong release mula sa [seksyon ng mga release](https://github.com/MCCTeam/Minecraft-Console-Client/releases/latest).

## Mabilis na Pag-install ⚡

Buksan ang terminal sa folder kung saan mo gustong ilagay ang MCC at patakbuhin ang:

Linux / macOS:

```bash
curl -fsSL https://mccteam.github.io/install.sh | sh
```

Windows (PowerShell):

```powershell
iwr -useb https://mccteam.github.io/install.ps1 | iex
```

Tinutukoy ng script ang iyong arkitektura at i-download ang tamang binary. Para sa iba pang mga opsyon (kabilang ang `wget` at manu-manong pag-download), tingnan ang [gabay sa pag-install](https://mccteam.github.io/guide/installation.html).
