# À propos

Minecraft Console Client (MCC) est un client TUI léger, cross-plateform et open-source pour Minecraft Java Edition qui vous permet de vous connecter à n'importe quel serveur Minecraft sous java, d'envoyer des commandes et de discuter dans le chat de manière rapide et simple sans avoir votre client Minecraft de lancé.

Il vous donne également accès à des fonctionnalités qui vous permettent d'administrer et de gérer votre serveur, mais aussi une API en C# pour vous offrir la possibilité de créer vos propres Bots.

Ce projet prend sa source d'un projet de ORelioOuvrir dans une nouvelle fenêtre de 2012 sur Minecraft ForumOuvrir dans une nouvelle fenêtre, maintenant il est maintenu par lui, mais aussi par de nombreux membres de la communauté.

# Fonctionnalités

  • Chat

    • Envoyer et recevoir des messages dans le chat
    • Enregistrer l'historique du chat
    • Recevoir des alertes en fonction de mots-clés
    • Réponses automatiques

  • Anti AFK

  • Reconnexion automatique

  • Planificateur de scripts

  • Contrôle à distance

  • Réponses automatiques

  • Attaque automatique

  • Pêche automatique

  • Manger automatiquement

  • Craft automatique

  • Mailer Bot

  • Drop automatique

  • Replay Mod

  • Une API pour créer des Bots en C#

  • Support de Docker

  • Gestion de l'inventaire

  • Traversement de terrain

  • Gestion des entités

Note : Certaines des fonctionnalités mentionnées sont désactivés par défaut et vous devrez les activer dans le fichier de configuration, de plus certaines fonctionnalités nécessiteront des configurations spécifiques de votre côté pour des utilisations spécifiques.

# Remarques sur certaines fonctionnalités

# Gestion de l'inventaire, du terrain et des entités

La gestion de l'inventaire n'est actuellement pas supportée pour les versions : 1.4.6 - 1.9 (* Le code de la gestion de l'inventaire est en place, mais il manque les palettes d'objets, sur lesquels nous travaillons.*)

La gestion du terrain n'est actuellement pas supportée pour les versions : 1.4.6 - 1.6

La gestion des entités n'est actuellement pas supportée pour les versions 1.4.6 - 1.7

Ces fonctionnalités ne seront pas toujours implémentées dans les dernières versions du jeu, qui sont bien souvent très différentes les unes des autres. Nous avons ainsi besoin de beaucoup de temps pour trouver ce qui a changé et implémenter les modifications nécessaires.

S'il y a eu une mise à jour majeure du jeu et que MCC n'a pas été mis à jour pour supporter les nouvelles fonctionnalités, si vous êtes développeur, vous êtes libres de contribuer au projet.

# Recherche de chemins et physique

Actuellement la recherche de chemin et la physique ont quelques limites, les voici :

  • La recherche de chemin sur les dalles n'est pas prise en charge (en cours de réalisation, partiellement terminer mais non disponible sur la branche principale)
  • La nage n'est pas encore prise en charge
  • Le saut n'est pas encore pris en charge
  • Le recul n'est pas encore pris en charge

Ce qui est actuellement supporté et fonctionne :

  • Le déplacement sur le terrain (recherche de chemin avec l'algorithme A* et la marche)
  • Monter et descendre les échelles et tous les types de vignes
  • Gravité

# Avertissement

Même si tout devait fonctionner, nous ne sommes responsables d'aucun dommage que cette application pourrait causer à votre ordinateur ou à votre serveur.

Cette application ne vole pas vos mots de passe. Si vous ne lui faites pas confiance, vérifiez là & compilez directement depuis le code source.

De plus, n'oubliez pas que quand vous vous connectez à un serveur avec ce programme vous apparaitrez là où vous vous êtes déconnecté la dernière fois.

Cela signifie que ** vous pouvez mourir si vous vous connectez dans un endroit dangereux sur un serveur survie !**

Utilisez le bot de planificateur de script pour envoyer une commande de téléportation après vous être connecté.

Nous vous rappelons que vous pouvez vous faire bannir par votre serveur en utilisant ce programme. À utiliser en accord avec les règles du serveur.
