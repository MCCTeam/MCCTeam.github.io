# 创建简单脚本

一个简单的脚本是一个包含每行一个命令的文本文件。 查看 “内部命令”在新窗口打开 部分或在控制台中输入 /help 以查看可用命令。 任何以 # 开头的行都会被忽略并作为注释处理。

使用 set 命令或 [AppVars] INI 部分定义的应用程序变量可以使用。 以下只读变量也可以使用：%username%，%login%，%serverip%，%serverport%，%datetime%

# 示例

sample-script.txt: 发送一条问候消息，等待 60 秒并从服务器断开连接。

# 这是 Minecraft 控制台客户端的示例脚本
# 以“#”开头的任何行都会被忽略并作为注释处理。

send 你好世界！ 我是使用 Minecraft 控制台客户端脚本编写的机器人。
wait 60
send 现在要退出了。 再见 :)
exit
