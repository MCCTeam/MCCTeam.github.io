# 关于

Minecraft 控制台客户端 (MCC) 是一个轻量，跨平台，开源的命令行界面 Minecraft 客户端，适用于国际Java版。您可以连接任何国际Java版服务器，用简单快速的方式发送指令和聊天，接收聊天，并且完全无须打开 Minecraft 游戏。

它也支持许多自动化功能，允许您轻松地管理和控制以及拓展 （基于C#）您的 Minecraft 机器人。

最初在2012年由ORelio在新窗口打开在Minecraft Forum在新窗口打开中创建,现在该项目由他和社区中的许多开发者支持与维护。

# 功能

  • 聊天

    • 发送和接收聊天信息
    • 记录聊天记录
    • 出现特定关键字时发出提醒
    • 自动回复

  • 反挂机检测

  • 自动重连

  • 脚本任务

  • 远程控制

  • 自动回复

  • 自动攻击

  • 自动钓鱼

  • 自动进食

  • 自动合成

  • 邮箱机器人

  • 自动丢弃

  • Replay模组

  • 基于C#的MCC Bot API

  • Docker 支持

  • 库存处理

  • 处理地形

  • 实体处理

注意:上面提到的一些功能需要在配置文件(MinecraftClient.ini)中打开,一些还需要您手动配置才能正常使用。

# 关于某些功能的注释

# 手持物品、地形和实体处理

物品栏操作当前不支持以下版本: 1.4.6 - 1.9 (物品栏操作的代码已经实装, 但是我们缺少物品调色盘，正在完善中。)

地形处理当前在以下版本中不受支持：1.4.6 - 1.6

实体处理当前在以下版本中不受支持：1.4.6 - 1.7

最新游戏版本可能无法使用这些功能，因为Mojang对游戏的大量修改。我们需要时间来去适配。

如果有新的游戏版本MCC不支持，我们欢迎任何有能力的人去帮助这个项目。

# 寻路与物理

目前，寻路与物理有一些局限性，它们是：

  • 半砖下的寻路功能不受支持(正在完善，部分分支完成，但主分支暂不支持)
  • 暂不支持游泳
  • 暂不支持跳跃
  • 暂不支持击退

已经支持并起作用的：

  • 地形导航(寻路算法使用 A* )
  • 上下攀爬梯子与所有类型的藤蔓
  • 下落

# 免责声明

即使一些正常工作，我们也不能保证它不会对服务器或者电脑造成伤害。

这个软件不会盗取您的账号密码。 如果您不信任它，那就不要使用它或者从源码检查有无恶意代码 。

记住: 如果您使用MCC连接到一个服务器, 您将会在上次下线的地方。

也就是说如果您在一个生存服务器，您可能会因为在一个不安全的地方而导致死亡！
