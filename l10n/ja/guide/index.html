# 概要

**Minecraft Console Client（MCC）**は、Minecraft を開かなくても、サーバーへ接続し、コマンドの送信、メッセージの受信などをすばやく簡単に行える、Java Edition 用の軽量クロスプラットフォームのオープン ソース Minecraft TUI クライアントです。

また、さまざまな目的で利用できる自動化機能と、Bot を拡張できる C# で記述された API も提供しています。

元々は 2012 年に Minecraft Forum新しいタブで開く で ORelio新しいタブで開く によって作成されましたが、現在は彼とコミュニティの多数の貢献者によって維持されています。

# 機能

  • チャット

    • チャット メッセージの送信 / 受信
    • チャット履歴の記録
    • 特定のキーワード検出時に通知
    • オート返信

  • 無操作回避

  • オート再接続

  • スクリプト スケジューラー

  • リモート コントロール

  • オート返信

  • オート攻撃

  • オート釣魚

  • オート イート

  • オート クラフト

  • メール Bot

  • オート ドロップ

  • Replay Mod

  • C# API

  • Docker サポート

  • インベントリ操作

  • 移動処理

  • エンティティ処理

注意: 一部の機能はデフォルトでオフになっており、設定ファイルでオンにする必要があります。

MCC はお互いに助け合うコミュニティです。GitHub リポジトリには Discussions セクションがあります。

こちら新しいタブで開くからアクセスしてください。

# 助けを求める前に

  • Discussions セクションとドキュメントで検索して、質問の重複を避けてください。 ご協力くださると幸いです。
  • 他の方が時間を使って助けてくれるので、親切に忍耐強く、他人を尊重し合ってください。

# バグ報告やアイデア / 機能リクエスト

バグ報告やアイデア / 機能リクエストは、GitHub リポジトリ新しいタブで開くの Issues新しいタブで開く セクションで行えます。

バグ報告やアイデア / 機能リクエストを送信する前に、Issues セクションとドキュメントで検索して重複している項目がないかご確認ください。

似たような項目が見つからない場合は、New issue ボタンをクリックして、フィードバックを送信してください。

バグを報告する場合は、バグを再現する方法の説明や、スクリーンショットやログの添付など、できるだけ詳しく説明してください。また、設定ファイルの DebugMessages が true になっていることをご確認ください。

# 送信する前に

  • 事前に Issues セクションとドキュメントで検索して、バグ報告やアイデア / 機能リクエストの重複を避けてください。 ご協力くださると幸いです。
  • 親切に忍耐強く、他人を尊重し合ってください。 ご協力くださると幸いです。

# 一部の機能に関する注意事項

# インベントリ操作や地形処理 / エンティティ処理

現在、インベントリ操作は次のバージョンではサポートされていません: <0>1.4.6</0> ～ <0>1.9</0>（インベントリ操作のコードは存在しますが、アイテム パレットが存在しません。）

現在、地形処理は次のバージョンではサポートされていません: 1.4.6 ～ <0>1.6</0>

現在、エンティティ処理は次のバージョンではサポートされていません: 1.4.6 ～ 1.7

最新バージョンでは Mojang によって大幅な変更が加えられ、機能しなくなることがあります。その場合は、実装にしばらく時間がかかります。

Minecraft のアップデートが MCC に対応していない場合、プログラマーであれば遠慮なくプロジェクトに貢献してください。

# Path-Finding and Physics

Currently the path-finding and physics have some limitations, those are:

  • Path finding under slabs is not supported (currently being worked on, partialy complete but not avaliable in the main branch)
  • Swimming is not supported yet
  • Jumping is not supported yet
  • Knockback is not supported yet

What is supported and works:

  • Terrain navigation (path-finding with A* algorithm and walking)
  • Climbing up and down the ladders and all types of vines
  • Gravity

# 免責事項

正常に動作したとしても、お使いのコンピューターやサーバーに損害が発生しないということを保証できません。

このアプリはパスワードを収集しません。 信頼できない場合は、ソース コードから確認してコンパイルするか、使用しないでください。

MCC を使用してサーバーに接続すると、最後に切断した位置に戻ります。

サバイバル サーバーで安全でない位置で切断すると、死亡してしまうことがあります。

スクリプト スケジューラーを使用してログインした際にテレポート コマンドを送信します。

MCC 使用すると、サーバーへのアクセスが禁止されるおそれがあります。 サーバーの規約をよく読んでから使用してください。
