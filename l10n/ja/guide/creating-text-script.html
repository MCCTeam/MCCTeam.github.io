# 簡単なスクリプトの作成

簡単なスクリプトは、1 行に 1 つのコマンドを書いたテキスト ファイルです。 利用できるコマンドを確認するには、内部コマンド新しいタブで開くをご覧になられるか、コンソールに /help と入力してコマンド リストを表示します。 # で始まる行は無視され、コメントとして扱われます。

set コマンドまたは [AppVars] INI セクションを使用して、定義されたアプリケーション変数を使用できます。 次の読み取り専用変数を使用することもできます: %username%、%login%、%serverip%、%serverport%、%datetime%

# サンプル

sample-script.txt: Hello メッセージを送信し、60 秒後にサーバーを退出します。

# これは Minecraft Console Client のサンプル スクリプトです。
# 「#」で始まる行は無視され、コメントとして扱われます。

send Hello World! これは Minecraft Console Client のスクリプト Bot です。
wait 60
send 退出します。またね～！ 
exit

その他のサンプルは、サンプル スクリプト新しいタブで開くをご確認ください。

高度な機能が必要な場合は、ChatBot を作成することもできます。
