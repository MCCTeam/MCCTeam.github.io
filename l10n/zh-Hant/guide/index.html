關於

Minecraft 主控台客戶端 (MCC) 是一个輕量級、跨平台、基於命令列介面的 Minecraft 客戶端，適用於 Java 版，允許你連接到任何 Minecraft Java 版伺服器，以快速簡單的方式發送命令與接收文字信息，而無需打開 Minecraft 遊戲。

它提供了各種各樣可以讓你管理及其他用途的自動化功能，以及用於編寫機器人的可拓展 C# 接口。

它最初是由 ORelio 在2012年製作，在 Minecraft Forum 發佈。現在由他以及許多來自社群的貢獻者維護。

特色功能

  • 聊天

    • 傳送和接收聊天信息
    • 匯出聊天紀錄
    • 出現特定關鍵字時得到提醒
    • 自動回覆信息

  • 反掛機踢出（Anti AFK）

  • 自動重連（Auto Relog）

  • 腳本排程（Script Scheduler）

  • 遠端控制（Remote Control）

  • 自動回覆（Auto Respond）

  • 自動攻擊（Auto Attack）

  • 自動釣魚（Auto Fishing）

  • 自動進食（Auto Eat）

  • 自動合成（Auto Craft）

  • 郵件機器人（Mailer Bot）

  • 自動丟棄（Auto Drop）

  • 回放模組

  • 用於編寫機器人的C# API

  • 支援 Docker

  • 庫存處理

  • 地形行走

  • 實體處理

注意：以上部分提到的功能預設為關閉狀態，你需要修改配置檔案來啟用它們；部分功能需要你針對特定用途進行額外配置。

部分功能的提示

庫存、地形及實體處理

Inventory handling is currently not supported in versions: 1.4.6 - 1.9 (The inventory handling code is in the place, but we're missing Item Palettes, on which we're working.)

地形處理不支援 1.4.6 - 1.6 版本

Entity handling is currently not supported in versions: 1.4.6 - 1.7

這些功能可能並不會在最新遊戲版本中實現，因為它們經常被 Mojang 作出大量修改。我們需要時間來研究哪些地方修改了和實現那些改變。

如果遊戲有版本更新而 MCC 又未更新至新版本，我們歡迎任何程序員為專案作出貢獻。

Path-Finding and Physics

Currently the path-finding and physics have some limitations, those are:

  • Path finding under slabs is not supported (currently being worked on, partialy complete but not avaliable in the main branch)
  • Swimming is not supported yet
  • Jumping is not supported yet
  • Knockback is not supported yet

What is supported and works:

  • Terrain navigation (path-finding with A* algorithm and walking)
  • Climbing up and down the ladders and all types of vines
  • Gravity

免責聲明

即使所有東西應該正常運作，對於此應用程序可能對你的電腦或伺服器造成的任何損害，我們概不負責。

這個程式不會偷取你的密碼。 如果你不信任這個程式，請不要使用，或是請自行檢查原始碼 & 編譯。

請記得當你使用這個程式連接伺服器時，你的角色將會出現在你上次登出時的最後位置。

這表示如果你在不安全的位置登入生存伺服器，你的角色可能會死亡。

使用腳本排程（script scheduler）在登入後自動傳送傳送指令（teleport command）。

我們提醒你，你可能會因使用此程式而被你的伺服器封禁。 按照伺服器的規則使用。
